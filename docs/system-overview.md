# 系统说明

## 业务定位

平台面向新疆智慧农业科研管理、特色作物富硒品质预测、土壤环境评估、抗逆栽培决策和项目展示场景。核心对象包括地块、作物品种、种植档案、土壤样本、气象记录、预测结果、评估结果、决策建议、报告和数据来源。

## 模块划分

| 模块 | 数据表 | 说明 |
| --- | --- | --- |
| 认证与用户 | `sys_user` | 登录、JWT、用户信息 |
| 地块管理 | `field_plot` | 地块编号、地区、经纬度、土壤类型、pH、有机质、硒、盐分、电导率、风险 |
| 品种与档案 | `crop_variety`、`crop_record` | 作物类型、品种、地块关联、播种采收、水肥、产量、品质 |
| 土壤与气象 | `soil_sample`、`weather_record` | 检测样本和气象记录 |
| 智慧分析 | `selenium_prediction`、`soil_assessment` | 富硒品质预测和土壤综合评分 |
| 决策与成果 | `decision_suggestion`、`report_record` | 动态建议和报告生成 |
| 数据治理 | `data_source`、`operation_log` | 数据来源和操作审计 |

## 富硒品质预测规则模型

模型版本：`XJ-Se-Explainable-0.2`

输入字段：

- 作物类型、品种名称、地块 ID
- 土壤硒含量、pH、有机质、盐分、电导率
- 平均温度、降水量
- 灌溉方式、施肥方式

核心逻辑：

- 土壤硒含量是主导因子。
- pH 越接近 6.8-8.2，硒吸收效率越高；过酸或过碱会降低吸收效率。
- 有机质在 16-28 g/kg 附近时提升预测结果，过低或过高会降低贡献。
- 盐分和电导率偏高时降低预测结果。
- 不同作物配置不同硒吸收系数：棉花、小麦、玉米、葡萄、枸杞、番茄分别采用不同系数。
- 气温和降水用于计算生长适宜性。
- 滴灌、水肥一体化、有机肥和富硒示范施肥会影响管理系数。

输出结果：

- 预测硒含量
- 等级：低硒、普通、富硒、高硒风险
- 可信度
- 影响因子贡献 JSON
- 影响因素解释
- 种植建议
- 输入签名和模型版本

同一输入得到稳定结果，不使用完全随机数。

## 土壤环境评估规则模型

输入指标：

- pH
- 有机质
- 土壤硒含量
- 盐分
- 电导率
- 重金属风险

评分方式：

- 每个指标转换为 0-100 的单项得分。
- 综合评分按权重合成：pH、有机质、土壤硒、盐分、电导率、重金属风险。
- 重金属高风险、盐分过高、电导率过高会直接抬升风险等级。

输出结果：

- 综合评分 0-100
- 风险等级：适宜、轻度预警、中度风险、高风险
- 单项指标评价 JSON
- 雷达图数据
- 综合评价说明
- 土壤改良建议

## 决策建议逻辑

用户选择地块和种植档案后，系统读取：

- 地块土壤环境指标
- 作物类型和水肥方式
- 最新土壤样本
- 最新富硒预测结果
- 最新土壤评估结果

动态生成五类建议：

- 富硒种植建议
- 土壤改良建议
- 灌溉施肥建议
- 风险预警建议
- 适宜推广区域建议

每条建议标记为推荐、谨慎或风险，并保存到 `decision_suggestion`。

## 报告生成逻辑

报告类型：

- 地块富硒种植适宜性报告
- 土壤环境综合评估报告
- 作物富硒品质预测报告
- 智慧农业综合决策报告

报告保存为数据库 HTML 内容，前端支持预览和浏览器打印或导出 PDF。报告内容包括基本信息、地块数据、作物数据、土壤评估、富硒预测、图表摘要、决策建议、数据来源和生成时间。

## 安全与工程约束

- 登录接口返回 JWT。
- `/api/**` 除登录外均需要 `Authorization: Bearer <token>`。
- 密码使用 BCrypt 哈希存储。
- 后端包含统一返回格式、参数校验、全局异常处理、CORS。
- MyBatis-Plus 使用参数化查询，避免拼接 SQL。
- 所有分页接口支持 `page/current` 和 `size`。
- Swagger/OpenAPI 地址：`/swagger-ui.html`。
