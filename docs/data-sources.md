# 数据来源与使用说明

本项目为科研训练、毕设和比赛展示型智慧农业系统。代码、界面、数据库结构和初始化脚本均为原创实现；未复制既有软件著作权系统的代码、页面文案或表结构。

当前开发环境没有执行在线采集任务，因此业务初始化数据为合理模拟数据，仅用于系统演示、算法联调和前端交互验证。所有模拟口径已写入 `data_source` 表，并在本文件标注。

## 初始化数据口径

| 数据表 | 数据性质 | 生成方式 | 用途 |
| --- | --- | --- | --- |
| `sys_user` | 系统测试数据 | 手工初始化，密码 BCrypt 哈希 | 登录和鉴权 |
| `field_plot` | 模拟数据 | 按新疆典型地区、经纬度、土壤类型和风险等级构造 30 条 | 地块管理、预测、评估和驾驶舱 |
| `crop_variety` | 模拟数据 | 按棉花、小麦、玉米、葡萄、枸杞、番茄构造合理品种名 | 种植档案选择 |
| `crop_record` | 模拟数据 | 按地块、品种、播种采收日期、水肥方式构造 50 条 | 种植档案和决策建议 |
| `soil_sample` | 模拟数据 | pH、有机质、氮磷钾、硒、盐分、电导率、重金属风险构造 100 条 | 土壤样本管理和土壤评估 |
| `weather_record` | 模拟数据 | 按新疆绿洲农业春夏季温度、降水、日照、辐射构造 90 条 | 气象图表和预测补齐 |
| `selenium_prediction` | 模拟/算法生成 | 种子示例加用户运行预测生成 | 首页趋势和预测历史 |
| `soil_assessment` | 模拟/算法生成 | 种子示例加用户运行评估生成 | 土壤评分、雷达图和风险分布 |
| `decision_suggestion` | 模拟/算法生成 | 种子示例加用户选择地块作物后动态生成 | 决策建议中心 |
| `report_record` | 模拟/系统生成 | 用户生成报告后保存 HTML 内容 | 报告预览和打印 |
| `data_source` | 说明数据 | 人工维护 | 记录来源、是否模拟和用途说明 |
| `operation_log` | 系统运行数据 | 后端拦截器写入 | 审计和系统设置页展示 |

## 模拟数据合理性约束

- 地区覆盖乌鲁木齐、昌吉、石河子、阿克苏、喀什、库尔勒、吐鲁番、哈密、伊犁。
- pH 主要位于新疆绿洲农业常见偏中性至偏碱范围。
- 土壤硒含量设置为低硒、中等硒背景、富硒背景和高硒风险的混合分布。
- 盐分、电导率和重金属风险用于形成适宜、轻度预警、中度风险、高风险分层。
- 气象数据采用新疆春夏季农业生产常见温度、降水、日照、辐射范围。
- 作物类型限定为棉花、小麦、玉米、葡萄、枸杞、番茄，并按新疆农业场景配置品种和水肥方式。

## 后续接入真实数据建议

- 统计数据：新增年度统计指标表，记录地区、作物、面积、产量、单产、年份、来源 URL 和口径说明。
- 土壤数据：接入实验室检测或采样点数据，保留采样坐标、检测机构、检测方法、批次和质量控制字段。
- 气象数据：通过 NASA POWER Daily API 或本地气象站接口定时拉取，记录经纬度、参数编码、时间范围和采集时间。
- 模型数据：将规则模型输出作为基线，后续接入机器学习模型服务，并保存模型版本、训练数据范围和验证指标。
